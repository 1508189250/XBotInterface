Detailed Description

The MatLogger class provides functionality to log numerical data to binary .mat files which can be easily imported in MATLAB/ OCTAVE/scipy/...

Usage:

  • obtain a pointer to the MatLogger by calling the factory method MatLogger::getLogger(filename_without_extension)
  • during the initialization phase, preallocate memory for the variables to be logged (not mandatory, but makes the logger RT safe) with methods createScalarVariable(), createVectorVariable(), createMatrixVariable()
  • inside the loop, log data with the method add()
  • you can actually dump data to the mat file manually by calling flush(), otherwise the dumping will be done inside the destructor
